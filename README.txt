  INSTALLATION GUIDE (For Non-Technical Users)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHAT YOU NEED:
  • A computer (Windows, Mac, or Linux)
  • A web browser (Chrome, Firefox, Edge — all work fine)
  • The KAIZEN folder (which you already have!)

HOW TO OPEN:
  METHOD 1 (Easiest — Double Click):
  1. Open the KAIZEN folder
  2. Double-click the file called "index.html"
  3. It will open in your default web browser
  4. Done! KAIZEN is ready to use.

  METHOD 2 (Drag and Drop):
  1. Open your web browser (Chrome, Firefox, etc.)
  2. Drag the "index.html" file into the browser window
  3. KAIZEN will open immediately

  METHOD 3 (Browser Address Bar):
  1. Open your web browser
  2. Press Ctrl+O (Windows) or Cmd+O (Mac)
  3. Navigate to the KAIZEN folder
  4. Select "index.html" and click Open

  FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. DRUG FILTER (⚗)
     - Paste up to 2000 SMILES compounds
     - Automatic Lipinski Rule of Five screening
     - ADMET property elimination
     - Download passed/failed lists as CSV

  2. COMPOUND SEARCH (🔍)
     - Search by name, SMILES, CID, or InChI
     - Get full molecular properties
     - Lipinski profile visualization
     - PubChem integration

  3. CONVERSION TOOL (⇄)
     - Convert between SMILES, CID, SDF, PDB, InChI
     - Batch convert up to 1000 compounds
     - Download as CSV, TXT, SDF, or PDB

  4. MOLECULAR INTERACTION (⚛)
     - Input SMILES + protein target
     - Visualize 2D molecular structure
     - Interaction analysis (H-bonds, hydrophobic, etc.)
     - Binding score estimation

  5. PLANT DATABASE (🌿)
     - Search plants by common/scientific name
     - Get all phytochemical compounds
     - Properties, activities, SMILES for each compound
     - Currently includes: Turmeric, Neem, Ginger, Ashwagandha

  6. CHEMISTRY GAME (🎮)
     - Molecule Match: Card-matching memory game
     - Chem Quiz: Test your pharmacology knowledge
     - Formula Builder: Identify chemical formulas

  7. AI ASSISTANT (🤖)
     - Ask questions about drug discovery
     - Lipinski, ADMET, SMILES explanations
     - Pharmacology concepts
     - Tool usage help

  TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Q: The page looks broken or plain
  A: Use Chrome or Firefox. Internet Explorer is not supported.

  Q: The 3D viewer doesn't show anything
  A: First enter a SMILES string, then click "Analyze Interaction"

  Q: The AI assistant doesn't respond
  A: Check your internet connection. The AI needs online access.

  Q: Conversion gives "[PubChem lookup]" results
  A: Some compounds need online lookup. Try: Aspirin, Caffeine,
     Quercetin, Curcumin, Morphine, Paracetamol, or Ibuprofen

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
